Реализация моего годового проекта (чейпойнт 4)

Структура репозитория:
year_project/
├── Dockerfile              # докер файл для запуска моего приложения в контейнере 
├── inference.py            # питон файл для реализии инференса йоло модели 
├── main.py                 # питон файл для работы с фаст апи
└── README.md               # описание проекта
└── requirements.txt        # список требуемых библиотек

В данном проекте мы запустили библиотеки, файлы для реализации инференса йоло модели, работа с фаст апи для запуска инференса в облаке, запуск приложения в контейнере.
